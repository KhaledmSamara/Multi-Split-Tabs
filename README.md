# Multi Split Tabs - Browser

A desktop browser with multiple tabs and multiple sessions. Each session can open in multiple windows. Easy to use.

## Features

- Multiple tabs in split screen (No Borders)
- Multiple sessions
- Open same session in multiple windows
- Save and load sessions
- Custom keyboard shortcuts
- Dark theme interface

## Download & Install

**Windows:** Download the setup file and run it
https://github.com/KhaledmSamara/Multi-Split-Tabs/releases/download/release-1/Windows_Multi.Split.Tabs.Setup.exe

## **Default Shortcuts** (can be changed in settings):

- **Add Tab**: Ctrl+T
- **Delete Tab**: Ctrl+W
- **Search**: Ctrl+F
- **Go Back**: Alt+Left
- **Go Forward**: Alt+Right
- **New Window**: Ctrl+N
- **New Session**: Ctrl+Shift+N
- **Switch Sessions**: Ctrl+Shift+Tab
- **Fullscreen**: F11
